# Solian-React

使用 React 19 + TypeScript + Vite + Ant Design 实现的 Solar Network 客户端。
由TRAE SOLO开发

## 技术栈

- **React 19** — 前端框架
- **TypeScript** — 类型安全
- **Vite** — 构建工具
- **Ant Design** — UI 组件库
- **React Router** — 路由管理
- **Zustand** — 状态管理
- **Axios** — HTTP 客户端

## 功能

- 多因子认证登录（Challenge-Based）
- 第三方 OAuth 登录（GitHub / Google / Apple）
- 每日签到与运势展示
- 侧边栏导航布局

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
